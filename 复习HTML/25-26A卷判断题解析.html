1 {∧, ∨} 是联结词完备集。 F（假）

为什么错

只用 ∧ 和 ∨ 无法表达否定 ¬。

任何仅含 ∧ 和 ∨ 的公式，当所有命题变元取值为 T 时，整个公式也取值为 T。永远得不到 F，所以不是完备集。

完整完备集 {¬, ∧, ∨}、{¬, ∧}、{¬, ∨}、{¬, →}、{↑}（与非）、{↓}（或非）

记法 最小完备集可以只有一个联结词！（{↑} 或 {↓}）

2 "闪光的未必是金子"符号化为 ¬∀x(F(x)→G(x))。 T（真）

逐词翻译

∀x(F(x)→G(x)) = "所有闪光的东西都是金子"（闪光→金子）

¬∀x(F(x)→G(x)) = "并非所有闪光的东西都是金子" = 闪光的未必是金子 ✅

🔗 等价变体：¬∀x(F(x)→G(x)) ⇔ ∃x(F(x)∧¬G(x)) = "存在闪光但不是金子的东西"
3 集合 X 上的任意一个划分，确定了 X 上的一个偏序关系。 F（假）

为什么错

划分确定的是 等价关系，不是偏序关系！

等价关系偏序关系
对应概念划分（等价类）哈斯图（层级的）
对称性✅ 对称❌ 反对称

划分 → 每个等价类内的元素互为等价关系。两个不同类的元素之间没有关系。而偏序关系要求反对称性——划分的关系是对称的，和反对称正好相反！

4 设 A 和 B 是有限集合，f: A→B，则 f 是双射当且仅当 |A| = |B|。 F（假）

为什么错

|A| = |B| 是双射的 必要条件，不是充分条件！

反例：A = {1, 2}，B = {a, b}，|A| = |B| = 2
f(1)=a，f(2)=a → 不是单射（1≠2 但 f(1)=f(2)）→ 不是双射

正确说法 有限集合间，f 是双射 ⇔ f 是单射且 |A| = |B|（或 f 是满射且 |A| = |B|）。

5 非空集合 A 上的恒等关系 IA 既是 A 上的等价关系也是 A 上的偏序关系。 T（真）

逐一验证

IA = {⟨x, x⟩ | x ∈ A}，所有有序对都在对角线上。

性质等价关系要求偏序关系要求IA 满足吗？
自反性✅✅✅ ∀x, ⟨x,x⟩∈IA
对称性✅—✅ ⟨x,y⟩∈IA⇒x=y⇒⟨y,x⟩∈IA
传递性✅✅✅ 显然
反对称性—✅✅ ⟨x,y⟩∈IA且⟨y,x⟩∈IA⇒x=y

全部满足 → 既是等价关系又是偏序关系 ✅

💡 恒等关系是"最小的"等价关系和"最小的"偏序关系。
6 二叉树的中序行遍法的访问顺序是：左子树，右子树，树根。 F（假）

题目描述的不是中序！

遍历方式顺序口诀
前序根 → 左 → 右根左右
中序左 → 根 → 右左根右 ← 这才是中序！
后序左 → 右 → 根左右根 ← 题目说的这个

题目写的"左子树，右子树，树根" = 后序遍历，不是中序遍历。

7 若 T 是一棵 n 阶无向树 (n≥2)，则 T 的边连通度是 n−1。 F（假）

概念混淆！

边连通度 λ(T) = 使图不连通需要删去的最少边数。

树中任意一条边都是桥，删去 1 条边就使图不连通。

∴ λ(T) = 1（不是 n−1！）

🔗 正确的事实：n 阶树有 n−1 条边，但边连通度是 1。
题目把"边数"和"边连通度"搞混了。
8 平面图 G 有 K₃ 和 K₄ 两个连通分支，则 G 的对偶图 G* 不是哈密顿图。 F（假）

G* 实际上是哈密顿图

G = K₃ ∪ K₄（两个分离的平面子图），它们的对偶通过"外部面"顶点连接成一个连通图。

K₃ 的对偶：2 个顶点（内面 + 外面），3 条平行边 → 有哈密顿回路

K₄ 的对偶即 K₄ 自身 → 有哈密顿回路

合并后 G*：外部面顶点把两部分的回路"串"起来 → 存在哈密顿回路 → 题目说"不是哈密顿图"是错的

💡 一般结论：不连通平面图的对偶图总是连通的（因为所有分支共享外部面）。
9 无向连通图 G 中所有顶点的度数都是偶数，则 G 是欧拉图。 T（真）

欧拉图的充要条件

无向图是欧拉图 ⇔ 所有顶点度数为偶数 + 连通

题目已经给了"连通"和"所有顶点度数为偶数" → 恰好是充要条件！

⚠️ 对比 24-25 A卷判断题第9题："无向图G中所有顶点的度数都是偶数，则G是欧拉图" → F（缺少"连通"条件！）
本题多加了一个"连通"，答案就变成 T 了。一词之差，迥然不同。
10 n 阶循环群的子群的阶都是 n 的正因子。 T（真）

拉格朗日定理 + 循环群的特殊性

拉格朗日定理：任何群，子群的阶一定整除群的阶。

循环群的加强版：n 阶循环群，对 n 的每个正因子 d，恰好存在一个 d 阶子群。

例如：6 阶循环群 ⟨a⟩：

n 的正因子 d1236
对应子群{e}⟨a³⟩⟨a²⟩⟨a⟩

所以子群的阶不仅是 n 的正因子，而且每个正因子都有子群。
