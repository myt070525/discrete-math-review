📐 有向图 · 邻接矩阵 · 通路计数

离散数学期末高频题型 — 三步从图到答案

题目套路

给一张有向图 → ① 写邻接矩阵 A → ② 算 A² 和 A³ → ③ 从矩阵读答案

一、看图写邻接矩阵

📖 定义

A = (aij)n×n，其中 aij = vi 到 vj 的有向边条数

aij从 第 i 行 对应的顶点出发到 第 j 列 对应的顶点

🎨 示例有向图

5个顶点的有向图，仔细观察每条箭头的方向和顶点编号

✍️ 逐对检查 → 写出 A

行 = 出边  |  列 = 入边

v₁→ [ 0 1 1 0 0 ] v₂→ [ 0 0 1 1 0 ] v₃→ [ 0 0 0 0 1 ] v₄→ [ 1 0 0 0 0 ] v₅→ [ 0 0 0 1 0 ] ↓ ↓ ↓ ↓ ↓ v₁ v₂ v₃ v₄ v₅
💡 快速检查：
第 i 行之和 = vi 出度  |  第 j 列之和 = vj 入度  |  全部加起来 = 总边数

二、核心定理 — Aᵏ 的含义

Ak 的 (i, j) 元素 = 从 vi 到 vj 长度为 k 的通路数

🧠 直观理解

A113 = 1 → 从 v₁ 到 v₃ 走1步：v₁→v₃（1条）
A214 = 1 → 从 v₁ 到 v₄ 走2步：v₁→v₂→v₄（1条）
A311 = 1 → 从 v₁ 回到 v₁ 走3步：v₁→v₂→v₄→v₁（1条回路）

三、计算 A² 和 A³

📐 矩阵乘法回顾

(A2)ij = Σk=1n Aik · Akj
即：A 的第 i 行 · A 的第 j 列

🔢 一步一步算 A²

A211 = 0·0 + 1·0 + 1·0 + 0·1 + 0·0 = 0
A214 = 0·0 + 1·1 + 1·0 + 0·0 + 0·1 = 1（即 v₁→v₂→v₄）

五、考试万能套路

📋 所有可能的设问 & 答案位置

设问答案位置
vi 到 vj 长度 = 1 的通路Aij
vi 到 vj 长度 = 2 的通路A2ij
vi 到 vj 长度 = 3 的通路A3ij
vi 到 vj 长度 ≤ k 的通路Aij + A2ij + ... + Akij

vi 到自身长度 ≤ k 的回路Aii + A2ii + ... + Akii
图中长度 = k 的通路总数Ak 的所有元素之和

六、易错点 & 防坑提醒

⚠️ 坑1：长度 ≤ k 的通路，要累加 A + A² + ... + Aᵏ
不要只取 Aᵏ！"≤"意味着所有小于等于的都要算上。
⚠️ 坑2：回路只看对角线 Aᵏᵢᵢ
回路 = 起点=终点，所以是 (i,i) 位置。(i,j) 位置（i≠j）不是回路。
⚠️ 坑3：矩阵乘法别算错
第 i 行 × 第 j 列，每次乘两个对应位置再求和。算完最好验算：出度入度对不对。

看图写A → 矩阵相乘得A² A³ → 数在哪个位置就回答哪个问题
