📊 偏序关系 · 哈斯图 · 上下确界

离散数学期末高频题型 — 从关系矩阵到求上下界

📝 题目

A = {a, b, c, d, e}，R 是 A 上的偏序关系，子集 B = {c, d, e}

(1) 列举 R  →  (2) 画哈斯图  →  (3) 求 B 的上下界和确界

一、从关系矩阵 → R 的集合表达式

📖 关系矩阵长这样（5×5，行列对应 a,b,c,d,e）

矩阵中 1 表示有序对存在，0 表示不存在
黄色高亮 = 对角线（自反性），说明 R 是偏序关系
💡 读矩阵口诀： 行 → 第一个元素，列 → 第二个元素。
比如第 2 行第 1 列 = 1 → <b, a> ∈ R（b 在 a 下面）

✍️ 逐个读出 12 个有序对

<a,a> <b,a> <b,b> <c,a> <c,c> <d,a> <d,b> <d,c> <d,d> <e,a> <e,c> <e,e>

二、从 R 画哈斯图（三步法）

哈斯图 = 去掉自环 + 去掉传递边 + 箭头朝上

1 第一步：去掉所有自环

自环 = 对角线上的元素（自反性）：<a,a> <b,b> <c,c> <d,d> <e,e>，全部删掉。

剩余： <b,a> <c,a> <d,a> <d,b> <d,c> <e,a> <e,c>

2 第二步：去掉传递边

如果 d→b 且 b→a，则 d→a 就是传递边，可以删掉。

边能删吗？理由
<d,a>✗ 删d→b→a 已经连通，d→a 是传递边
<d,a>✗ 删也可走 d→c→a
<e,a>✗ 删e→c→a 已经连通

3 第三步：只保留覆盖关系 → 哈斯图

覆盖边： <b,a> <c,a> <d,b> <d,c> <e,c>
绿色虚线框 = 子集 B = {c, d, e}

三、求 B = {c, d, e} 的上界·下界·确界

🧭 先看 B 中三个元素之间的关系

c 在 d 和 e 的上方 · d 和 e 不可比

⬆️ 求上界 & 上确界

上界 = 比 B 中所有元素都大的元素

候选≥ c ?≥ d ?≥ e ?是上界?
a✅✅✅✅ 是
b不可比✅不可比✗
c✅✅
(d≤c)		✅
(e≤c)		✅ 是
d✗✅✗✗
e✗✗✅✗
📋 上界 = {a, c}
🎯 上确界 = c（上界中最小的：c ≤ a，c 更小）

⬇️ 求下界 & 下确界

下界 = 比 B 中所有元素都小的元素

需要同时 ≤ c、≤ d、≤ e：

  • ≤ d 的元素：只有 d 自己（d 已在最底层）
  • ≤ e 的元素：只有 e 自己（e 已在最底层）
  • 同时 ≤ d 且 ≤ e 的元素不存在！
📋 下界 = ∅（无）
🎯 下确界 = 不存在

四、答案总结 & 记忆口诀

问题答案
(1) R 的集合表达式{<a,a>,<b,a>,<b,b>,<c,a>,<c,c>,<d,a>,<d,b>,<d,c>,<d,d>,<e,a>,<e,c>,<e,e>}
(2) 哈斯图见上方图示（a在上，d最下）
(3) 上界a, c
(3) 上确界c
(3) 下界无
(3) 下确界无
🎯 考场速记：
哈斯图三去：去自环 → 去传递 → 箭头朝上
上界：比所有都大 · 下界：比所有都小
确界：界中取最值（上确界=最小上界，下确界=最大下界）
