💥 平面图证明 · 反证法

离散数学期末压轴题 — 24-25 A卷 证明题第3题

📝 题目

面数 r < 20 的简单连通平面图 G 中
必定存在顶点 v，其度数 d(v) ≤ 4

一、反证法思路 — 先听懂"为什么要反证"

🎯 这道题在说什么？

平面图 = 可以画在平面上且边不交叉的图
r = 面的个数（包括最外面的无限面）
💡 反证法思路：
直接证"存在 d(v)≤4 的顶点"很难 → 反过来想：
假设所有顶点度数都 ≥ 5，然后推导出 r ≥ 20，与已知 r < 20 矛盾！
矛盾说明假设错了 → 必定存在 d(v) ≤ 4 的顶点 ✅

二、先备好三把"工具"

🔧 工具①：握手定理

∑ d(v) = 2m

所有顶点度数之和 = 边数的 2 倍（每条边被两个端点各算一次）

🔧 工具②：欧拉公式（连通平面图）

n － m ＋ r ＝ 2

n=顶点数   m=边数   r=面数

🔧 工具③：简单图的面-边不等式

2m ≥ 3r

简单图（无重边无自环），每个面至少由 3 条边 围成
每条边至多被 2 个面 共享 → 2m ≥ 3r

三、逐步推导（跟着走一遍）

⚡ 出发点：反设

假设：G 中所有顶点度数 d(v) ≥ 5

1 握手定理 → n 和 m 的关系

已知 ∑d(v) = 2m，又每个 d(v) ≥ 5：

2m = ∑ d(v) ≥ 5n → n ≤ 2m/5　...... ①
5个顶点，全部d(v)≥5，总度数≥25，m≥12.5→m≥13

2 欧拉公式变形 → 用 m 表达 r

由 n − m + r = 2，移项得：r = m − n + 2

将 ① n ≤ 2m/5 代入（注意负号会使不等号翻转）：

r = m − n + 2 ≥ m − 2m/5 + 2 = 3m/5 + 2　...... ③

这给出了 r 的下界（r 至少是 3m/5+2）

3 面-边不等式 → r 的上界

简单平面图：2m ≥ 3r

r ≤ 2m/3　...... ②

这给出了 r 的上界（r 最多是 2m/3）

4 🔥 夹逼！上界 vs 下界

r 被夹在中间：

3m/5 + 2 ≤ r ≤ 2m/3

取下界 ≤ 上界：

3m/5 + 2 ≤ 2m/3

两边 ×15：9m + 30 ≤ 10m

→ 30 ≤ m →

m ≥ 30

如果所有 d(v)≥5，则图至少有 30 条边！

5 代回求 r → 矛盾！

将 m ≥ 30 代入下界：

r ≥ 3×30/5 + 2 = 18 + 2 = 20
💥 r ≥ 20   vs   已知 r < 20   矛盾！

✅ 结论

假设"所有 d(v) ≥ 5"不成立

∴ G 中必定存在顶点 v，使得 d(v) ≤ 4

四、完整推导链条（一张图看完）

五、同类题变式 & 通用模板

🔄 变式1：改"d(v)≤4"为"d(v)≤3"

题目：面数 r < 12 的简单连通平面图 G 中，必存在顶点 v 满足 d(v) ≤ 3。

解法：反设所有 d(v) ≥ 4 → 2m ≥ 4n → n ≤ m/2
→ r = m-n+2 ≥ m - m/2 + 2 = m/2 + 2
→ 与 r ≤ 2m/3 夹逼 → m/2+2 ≤ 2m/3 → m ≥ 12 → r ≥ 8
若把条件改小（r<8或r<12），就可以控制结论中 d(v)≤3 还是 ≤4

🔄 变式2：彼得森图不是平面图

25-26 A卷证明题第3题就是这个！

彼得森图：n=10, m=15, 无三角（最短圈长度=5）
若为平面图，则每个面 ≥ 5 条边 → 2m ≥ 5r → r ≤ 2m/5 = 6
欧拉公式：r = m-n+2 = 15-10+2 = 7
7 ≤ 6 矛盾！ → 彼得森图不是平面图

📋 万能反证法模板

步骤操作
① 反设假设"所有顶点都不满足结论"
② 握手∑d(v) = 2m → 推出 n 和 m 的不等式
③ 欧拉n − m + r = 2 → 用 m 表达 r
④ 面-边2m ≥ k·r（k=最小面边界数）→ r 的上界
⑤ 夹逼下界 ≤ r ≤ 上界 → 推出 m 的不等式
⑥ 回代m 代回 r 的表达式 → 与已知条件矛盾
⑦ 结论假设不成立 → 原命题得证

六、易错点

#易错点正确做法
1 欧拉公式写成 n+m-r=2 n − m + r = 2（牢记：点−边+面=2）
2 代入 n≤2m/5 时忘了负号翻不等号 r = m − n + 2，n 前是负号，n ≤ 某值 → −n ≥ −某值
3 面-边不等式记错：3r ≤ 2m 还是 3r ≥ 2m 记住：边数不能太少 → 2m ≥ 3r（边数×2 ≥ 面数×3）
4 非简单图也用 2m ≥ 3r 只有简单图才能用 3；若有重边或自环，下限不同
5 只推到 m≥30 就结束 必须回代求 r，矛盾点在 r 而不是 m
🎯 考场速记：
反设 d(v)≥5 → 握手得 n≤2m/5 → 欧拉 + 面-边夹逼 → m≥30 → r≥20 → 矛盾！
