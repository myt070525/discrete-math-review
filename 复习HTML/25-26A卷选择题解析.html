1 命题符号化 — "除非…否则…" 答案：A

设 p：天下大雨，q：他在室内运动。命题 "除非天下大雨，否则他不在室内运动"。

✅ A. p → q
B. q → p
C. q → ¬p
D. p → ¬q
🔑 关键公式：
"除非 p，否则 q" ⇔ "¬p → q" ⇔ "p ∨ q"

本题："除非天下大雨，否则他不在室内运动"
→ ¬p → ¬q （如果天没下大雨，则他不在室内运动）
→ 等价于 p → q（逆否命题）
→ 等价于 "如果天下大雨，则他在室内运动"
💡 记忆技巧："除非A否则B" = "如果不A则B" = ¬A → B。
本题 B = ¬q，所以 ¬p → ¬q，等价于 p → q。
2 命题公式类型判定 答案：B

公式：(p ∧ (p → q)) → q

A. 矛盾式
✅ B. 重言式
C. 非重言式的可满足式
D. 无法判定
🔑 化简：
p ∧ (p → q) ⇔ p ∧ (¬p ∨ q) ⇔ (p ∧ ¬p) ∨ (p ∧ q) ⇔ p ∧ q

原式 ⇔ (p ∧ q) → q
⇔ ¬(p ∧ q) ∨ q ⇔ (¬p ∨ ¬q) ∨ q ⇔ ¬p ∨ T ⇔ T

真值恒为T → 重言式（永真式）
💡 这就是"假言推理"的有效性：(p ∧ (p→q)) → q 是重言式，说明假言推理是有效的推理规则。
3 幂集的基数 + 笛卡尔积 答案：A

A = {a, b}，P(A) 是 A 的幂集，求 |A × P(A)|。

✅ A. 8
B. 16
C. 12
D. 无法判定
🔑 分两步：
① |A| = 2，P(A) = {∅, {a}, {b}, {a,b}}，|P(A)| = 2² = 4
② |A × P(A)| = |A| × |P(A)| = 2 × 4 = 8

展开验证：A × P(A) = {⟨a,∅⟩,⟨a,{a}⟩,⟨a,{b}⟩,⟨a,{a,b}⟩,⟨b,∅⟩,⟨b,{a}⟩,⟨b,{b}⟩,⟨b,{a,b}⟩}，共 8 个。
⚠️ 陷阱：|A × B| = |A| × |B| = 2 × 4 = 8，不要误算成 2⁴=16！
2⁴=16 是 |A→B|（函数个数）或混淆了笛卡尔积和幂集运算。
4 谓词逻辑等值式 答案：D

下列谓词公式不等值的是（注意题目问的是"不等值"！）。

A. ¬∀xA(x) ⇔ ∃x¬A(x)
B. ∀x¬A(x) ⇔ ¬∃xA(x)
C. ¬∃x(M(x)∧¬F(x)) ⇔ ∀x(M(x)→F(x))
✅ D. ¬∀x(F(x)→G(x)) ⇔ ∃x(F(x)∧G(x))
🔑 逐项验算：

A ✅（等值）：量词否定律：¬∀ = ∃¬
B ✅（等值）：量词否定律：∀¬ = ¬∃
C ✅（等值）：左边 ¬∃x(M(x)∧¬F(x)) ⇔ ∀x¬(M(x)∧¬F(x)) ⇔ ∀x(¬M(x)∨F(x)) ⇔ ∀x(M(x)→F(x)) = 右边
D ❌（不等值！）：正确公式是
¬∀x(F(x)→G(x)) ⇔ ∃x(F(x) ∧ ¬G(x))
题目D写的是 ∃x(F(x)∧G(x))，缺了 ¬！
⚠️ 考试大坑：题目问的是"不等值"，不是"等值"！
别急着勾第一个等值的，要找那个"错误"的。D 漏了 ¬G(x)，所以不等值。
5 3-正则图求 n 和 m 答案：A

n 阶无向图 G 是 3-正则图，m 为边数，且 2n − 3 = m。求 n 和 m。

✅ A. n = 6，m = 9
B. n = 5，m = 7
C. n = 3，m = 3
D. n = 7，m = 5
🔑 3-正则图：每个顶点度数为 3

握手定理：∑d(v) = 3n = 2m → m = 3n/2
结合条件：2n − 3 = m

联立：2n − 3 = 3n/2
→ 4n − 6 = 3n
→ n = 6
→ m = 2×6 − 3 = 9
→ 验证：3n/2 = 18/2 = 9 ✓
💡 k-正则图通用公式：∑d(v) = kn = 2m → m = kn/2。
k 必须是偶数或 n 必须是偶数（因为 m 必须是整数）。
6 关系幂运算 — 周期规律 答案：C

R 是 A 上的关系，已知 R³ = R⁶，求 R⁴⁴。

A. R³
B. R⁶
✅ C. R⁵
D. R⁰
🔑 从 R³ = R⁶ 推出周期：
R³ = R⁶ = R³ ∘ R³
两边合成 R³：R⁶ = R⁹
再合成：R⁹ = R¹² ...
→ 从 R³ 开始，每 3 步一个循环：R³ = R⁶ = R⁹ = R¹² = ...

R⁴⁴ = R³⁺⁴¹ = R³ ∘ R⁴¹
找 44 除以 3：44 = 3×14 + 2，不行，因为周期从 R³ 开始
关键：R³ = R⁶，则对于 k ≥ 3，Rᵏ = Rᵏ⁺³

R⁴⁴ = R⁴⁴⁻³ = R⁴¹ = ... = R³⁺² = R⁵（不断减3直到落在 [3,5] 内）
44 − 3×13 = 44 − 39 = 5 → R⁴⁴ = R⁵
💡 更简单的方法：R³=R⁶ ⇒ R³=R³∘R³ ⇒ 对任意k≥3，Rᵏ=R³∘Rᵏ⁻³。
当 k≥6 时 Rᵏ=Rᵏ⁻³。反复减 3：44→41→38→...→5。
7 函数复合 — f(x)=x², g(x)=x+2 答案：A

f: R→R, g: R→R，f(x)=x²，g(x)=x+2，求 f∘g 的性质。

✅ A. 既不是单射也不是满射
B. 是单射但不是满射
C. 是满射但不是单射
D. 是双射
🔑 先求 f∘g(x)：
(f∘g)(x) = f(g(x)) = f(x+2) = (x+2)²

单射判断：(x₁+2)² = (x₂+2)² 是否 ⇒ x₁=x₂？
取 x₁=−4 (得 4)，x₂=0 (得 4)，值相同但 x₁≠x₂ → 不是单射

满射判断：值域 = [0, +∞) ≠ R（取不到负数）→ 不是满射

→ 既不是单射也不是满射
8 K₄ 含3条边的生成子图（不同构） 答案：C

4 阶完全无向图 K₄ 中含 3 条边的不同构的生成子图有几个？

A. 5
B. 2
✅ C. 3
D. 以上都不对
🔑 生成子图 = 顶点全保留，只选部分边。K₄ 有 6 条边，选 3 条。

选 3 条边的所有可能结构（不同构！）：

① 3 条边构成一条路径 P₄：●━●━●━● （不闭合的链）
② 3 条边构成一个三角形 K₃ + 1 孤立点：△ + ●（3 条边在一个三角形上）
③ 3 条边构成星形 K₁,₃：一个中心点连三个叶子

一共 3 种不同构 的结构。
（注意：3条边不能构成含回路的 4 顶点结构除了三角形——回路需至少 3 条边，含 4 个点的回路需 4 条边。）
9 自同态 — V = ⟨R*, ·⟩ 答案：D

V = ⟨R*, ·⟩（非零实数乘法群），下列哪个是 V 的自同态？

A. f(x) = −x
B. f(x) = 2x
C. f(x) = x＋1
✅ D. f(x) = |x|
🔑 自同态条件：f(x·y) = f(x)·f(y)（保持运算）

A：f(x·y) = −xy，但 f(x)·f(y) = (−x)(−y) = xy ≠ −xy  ✗
B：f(x·y) = 2xy，但 f(x)·f(y) = 4xy ≠ 2xy  ✗
C：f(x·y) = xy+1，但 f(x)·f(y) = (x+1)(y+1) = xy+x+y+1  ✗
D：f(x·y) = |xy| = |x|·|y| = f(x)·f(y)  ✓

→ |x| 保持乘法运算
💡 同态 vs 自同态：同态 = 保持运算的映射。自同态 = 到自身的同态。
判断就是验证 f(x∘y) = f(x)∘f(y) 是否恒成立。
10 群的定义 — 哪个是群？ 答案：C

下列代数系统中，哪个是群？

A. ⟨N, ＋⟩
B. ⟨Z, ×⟩
✅ C. ⟨Zₙ, ⊕⟩
D. ⟨P(S), ∩⟩
群 = 封闭 + 结合 + 幺元 + 每个元素有逆元

A ⟨N,＋⟩：自然数加法，没有逆元（如 1 的逆元 −1 ∉ N） ✗
B ⟨Z,×⟩：整数乘法，除 ±1 外没逆元（如 2⁻¹ = 1/2 ∉ Z） ✗
C ⟨Zₙ,⊕⟩：模 n 加法群！封闭✓ 结合✓ 幺元0✓ 逆元n−a✓  ✓
D ⟨P(S),∩⟩：交运算，一般没有逆元（A∩B=∅ 不能恢复） ✗

→ ⟨Zₙ, ⊕⟩ 是群（模 n 加法群）
⚠️ 常见群和非群对比：
是群：⟨Z,＋⟩、⟨R*,×⟩、⟨Zₙ,⊕⟩
不是群：⟨N,＋⟩（无逆元）、⟨Z,×⟩（无逆元）、⟨P(S),∩⟩（无逆元）
